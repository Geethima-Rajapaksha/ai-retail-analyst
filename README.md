
🛍️ AI-Powered Retail Business Analyst

An intelligent business analytics tool that transforms 
raw retail sales data into actionable insights.

## 🚀 Features
- 📊 Automatic KPI calculation (Revenue, Profit, Growth)
- 📈 6 Interactive charts powered by Plotly
- 🤖 AI-generated business insights via Claude API
- 📄 Auto-export beautiful HTML reports
- 🗺️ Regional & category performance analysis

## 🛠️ Built With
- Python & Pandas — data processing
- Plotly — interactive visualizations
- Claude AI — intelligent insights
- Google Colab — development environment

## 📊 Dataset
- 5,017 orders across 2 years (2023-2024)
- 18 products across 4 categories
- 5 regions, 8 sales reps, 4 sales channels
